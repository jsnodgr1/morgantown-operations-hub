# Source Inventory

Use this document to record the workbook and tab inventory before logic is finalized in code.

## Source Template

| Field | Description |
| --- | --- |
| Source name | Friendly business name of the source |
| Owner | Team or individual responsible for the file |
| Business purpose | Why the source exists operationally |
| File path or location | Local path, shared drive, or managed location |
| Refresh cadence | Expected update frequency |
| Authoritative fields | Which fields this source is trusted for |
| Tabs/sheets used | Exact sheet names in scope |
| Timestamp fields | Source timestamp columns or workbook cells |
| Known manual edits | Manual interventions that may affect trust |
| Data quality concerns | Missing values, merged cells, formula issues, naming drift |
| Precedence notes | When this source should win or lose conflicts |
